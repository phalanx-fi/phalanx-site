# Fee Structure

Phalanx charges a single fee: a **10% performance fee** on positive yield above a high water mark.

| Fee Type | Rate |
|----------|------|
| Performance fee | 10% of yield above HWM |
| Management fee | None |
| Deposit fee | None |
| Withdrawal fee | None |

Fees are collected via share dilution. MAX_PERFORMANCE_FEE hardcoded at 20%.
